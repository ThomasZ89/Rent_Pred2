Overview

On this page you can find an overview of the features that have been extracted in order to predict the rent and what their meaning is. Further below is a detailed pandas profiling report where more information about each feature in the dataset is displayed.

Before using the data outliers have been filtered out. Therefore only entries with rent between 150 € and 1.200 € and room size between 6 and 60 m² are included. Furthermore rare or non existing categorical variables have been replaced with a dummy (no_info_or_rare).

In addition to that some features are engineered. Like "dauer" is the difference between start and end time. And "status" shows, if the advertisement was active at the time it was scraped.
