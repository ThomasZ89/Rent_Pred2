Model Overview

For the prediction with metadata the gradient boosting library Catboost and for the prediction based on text a Support Vector Machine from sklearn has been used.

Catboost

Currently the model hast the following specification:

CatBoostRegressor(iterations=4000, depth=12, loss_function='RMSE', early_stopping_rounds=30)

This results in a RMSE of 66.45 in the validation set (25% of data). Which means the average estimation is about 66 Euros off.

In general the RMSE seems to be pretty high, which means there are a lot of factors that are not explained by the metadata. Looking at some observations in detail, often times we can find missing or even false data (for example the rent is only on a weekly basis).
